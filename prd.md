# PRD: MBTI 관계 솔루션 서비스

## 문서 정보
- **버전**: 1.0
- **작성일**: 2024-12
- **작성자**: Ekan
- **목적**: 학원 프로젝트 → 실제 출시

---

## 1. 서비스 개요

### 1-1. 핵심 가치 제안

MBTI 유형 기반으로 대인관계 갈등을 분석하고 실행 가능한 해결책을 제시하며, 상대방에게 효과적으로 전달할 수 있는 맞춤 메시지를 생성하는 AI 상담 플랫폼

**한 문장 정의**  
"AI와 대화하며 MBTI 기반 관계 고민을 해결하고, 바로 쓸 수 있는 메시지를 받는 서비스"

### 1-2. 타겟 사용자

- **주 타겟**: 20-40대 직장인
- **부 타겟**: 대학생, 학원 강사, 팀 리더
- **페르소나**:
    - 직장에서 상사/동료와 갈등 있는 사람
    - MBTI에 관심 많고 실용적 해결책 원하는 사람
    - "뭐라고 말해야 할지 모르겠어" 고민하는 사람

### 1-3. 핵심 차별화

1. **자연스러운 대화형 상담** - 폼 입력이 아닌 AI와 채팅
2. **감질나는 무료 체험** - 2-3턴 vs 10-20턴 명확한 차이
3. **128가지 정밀 분석** - 7-letter 시스템 (업계 유일)
4. **즉시 활용 가능** - 복사해서 바로 보낼 수 있는 메시지
5. **원스톱 솔루션** - 분석 → 조언 → 메시지 작성까지

## 2. 핵심 기능

### 2-1. MBTI 상담소 (대화형)

#### 기능 개요
AI와 자연스러운 대화를 통해 관계 고민을 상담하고 MBTI 기반 분석 및 해결책을 제공

#### 초기 설정
- 사용자의 성별, MBTI는 자동 적용 (온보딩 시 입력)
- 매번 입력할 필요 없음

#### 대화 플로우

**Step 1: 대화 시작**
```
AI: "안녕하세요! 어떤 관계 고민이 있으세요?"
사용자: [자유롭게 입력]
```

**Step 2: AI 질문 (자연스러운 대화)**
```
AI: 상황 파악을 위한 질문
사용자: 답변
AI: 추가 질문 (맥락에 따라)
사용자: 답변
...
```

**Step 3: 대화 종료**
```
AI: "충분히 이해했어요! 지금 정리해드릴게요"
```

**Step 4: 종합 분석 결과**

4개 섹션으로 구성:

1. **상황 분석**
    - 대화 내용 종합
    - MBTI 관점에서 문제 재해석
    - 핵심 갈등 원인

2. **유형별 특성**
    - 내 MBTI 특성 + 행동 패턴
    - 상대방 MBTI 특성 + 행동 패턴 (알고 있는 경우)
    - 두 유형 간 차이점
    - 프리미엄: 7-letter 세밀 분석

3. **해결 방안**
    - 실행 가능한 액션 3-5개
    - 우선순위 표시
    - 각 방안의 기대 효과
    - 실제 대화 예시

4. **주의사항**
    - 피해야 할 말/행동
    - 상대방 민감 포인트
    - 역효과 가능성

#### 무료 vs 프리미엄 차이

**무료 (2-3턴 대화)**
- 턴 제한: 2-3턴 후 자동 종료
- 분석: 4-letter 기본 (16가지)
- 경험: "더 얘기하고 싶은데..." 감질
- UI: 턴 카운트 명확히 표시

```
[대화 UI 상단]
┌─────────────────────────┐
│ 무료: 대화 2턴 남음     │
│ 프리미엄: 깊은 대화 가능│
└─────────────────────────┘
```

**프리미엄 (10-20턴 대화)**
- 턴 제한: 10-20턴 (AI 판단 or 최대 20턴)
- 분석: 7-letter 정밀 (128가지)
- 경험: 충분히 깊은 대화, 완전한 이해
- UI: 턴 카운트 표시 안 함 (편하게)

```
[대화 UI 상단]
┌─────────────────────────┐
│ 프리미엄: 편하게 대화하세요│
└─────────────────────────┘
```

#### AI 대화 가이드라인

**AI 페르소나**
- 10년 경력 MBTI 전문 상담사
- 따뜻하고 공감적
- 경청하며 핵심 파악

**대화 스타일**
- 공감적 리액션: "힘드시겠어요 😔", "이해해요"
- 자연스러운 질문: 폼 느낌 아님
- 이모지 적절히 사용
- 맥락 따라 조절 (중요한 건 깊게, 덜 중요한 건 스킵)

**정보 수집 목표**
1. 갈등 상황 구체화
2. 빈도 및 심각도
3. 시도한 해결책 (있는 경우)
4. 감정 상태
5. 관계 맥락 (기간, 다른 사람과는 등)
6. 상대방 특성

**종료 조건**
- 무료: 3턴 강제 종료
- 프리미엄: AI가 충분한 정보 수집 판단 OR 20턴 도달

#### CTA (Call-to-Action)

**결과 페이지 하단**
- "이 조언을 바탕으로 메시지 작성하기" 버튼
- 클릭 시 변환기로 이동 (컨텍스트 자동 전달)

### 2-2. MBTI 대화 변환기

#### 기능 개요
전달하고 싶은 내용을 상대방 MBTI에 맞게 재작성

#### 2-2-1. 한마디 변환 모드 (기본)

**입력**
- 자동: 내 성별, 내 MBTI
- 사용자 입력:
    - 상대방 성별 (선택)
    - 상대방 MBTI (필수)
    - 전달할 내용 (20-500자)

**출력**

3가지 톤 버전으로 생성:

1. **공손한 버전**
    - 사용 상황: "회의에서", "이메일로", "윗사람에게"
    - 특징: 격식, 존댓말, 완곡 표현
    - 예시 제공

2. **캐주얼한 버전**
    - 사용 상황: "점심 먹으며", "메신저로", "친한 관계"
    - 특징: 친근한 대화체, 자연스러움
    - 예시 제공

3. **간결한 버전**
    - 사용 상황: "급할 때", "바쁜 상대", "짧게 전달"
    - 특징: 핵심만, 결론부터
    - 예시 제공

**각 버전 구성**
```
[메시지 텍스트]

[복사 버튼]

[해설]
"왜 이 표현이 ESTP에게 효과적인가" (2-3줄)
```

#### 2-2-2. 대화 컨텍스트 모드 (프리미엄 전용)

**입력**
- 기존 대화 전체 (카톡/이메일 복붙)
- 하고 싶은 말

**출력**
- 대화 흐름 파악 후 3가지 톤 버전
- 이전 대화 톤 유지
- 상대방 감정 상태 고려
- 갑작스럽지 않은 자연스러운 연결

**예시**
```
[입력]
기존 대화:
상사: "보고서 언제 주나?"
나: "내일 드리겠습니다"
상사: "급한데"

하고 싶은 말:
"오늘 중으로 드리겠습니다"

[출력]
대화 흐름상 상사가 조급해하는 상태 파악
→ 3가지 버전 생성 (상사의 급한 상황 고려)
```

### 3-2. 프리미엄 (회원)

#### 가입 정보

**필수**
- 이메일
- 비밀번호
- 닉네임
- 성별
- MBTI 7-letter

**MBTI 7-letter 구성**
```
[기본 4-letter]
16가지 중 선택

[확장 3가지]
1. A/T
   - A (Assertive): 자신감, 스트레스 덜 받음, 독립적
   - T (Turbulent): 신중함, 완벽주의, 타인 평가 민감

2. O/S
   - O (Open): 변화 선호, 새로운 시도 즐김, 유연
   - S (Stable): 안정 추구, 익숙한 것 선호, 체계적

3. R/I
   - R (Rational): 논리적 분석 후 결정, 신중한 판단
   - I (Impulsive): 직관 따름, 빠른 결단, 즉흥적
```

**각 항목 입력 지원**
- 설명 툴팁 제공
- "잘 모르겠어요" 선택 가능
- → "간단 진단하기" 버튼 (10-15개 질문으로 자동 판별)
- 나중에 마이페이지에서 수정 가능

#### 이용 혜택

- **무제한 사용**: 횟수 제한 없음
- **깊은 대화**: 10-20턴
- **정밀 분석**: 7-letter 기반 128가지 조합
    - 예: INTJ-AOI (혁신적 전략가) vs INTJ-TSR (신중한 계획가)
    - 같은 INTJ여도 완전히 다른 조언
- **변환기**: 한마디 + 대화 컨텍스트 모드 모두
- **히스토리**: 무제한 저장 및 검색
- **추천**: 과거 비슷한 상담 AI 추천
- **즐겨찾기**: 자주 상담하는 관계 조합 저장

#### 사용자 경험

**대화 중**
```
[턴 제한 없음]
"프리미엄: 편하게 대화하세요"

AI가 충분히 이해할 때까지 자유롭게 대화
```

**마이페이지**
```
- 내 정보 수정 (7-letter 포함)
- 상담 히스토리
  - 날짜별 리스트
  - 검색 기능
  - 상담 재열람
- 즐겨찾기 관리
```

## 6. 차별화 포인트

### 6-1. vs 기존 MBTI 서비스

| 항목 | 기존 서비스 | 우리 서비스 |
|------|------------|------------|
| 입력 방식 | 폼 입력 | AI와 대화 |
| 분석 | 16가지 (4-letter) | 128가지 (7-letter) |
| 결과 | 추상적 설명 | 실행 가능한 조언 |
| 메시지 | 없음 | 3가지 톤 자동 생성 |
| 경험 | 일방적 | 인터랙티브 |

### 6-2. 핵심 강점

**1. 자연스러운 대화**
- 폼 채우는 느낌 X
- 친구한테 털어놓듯이
- 심리적 허들 낮음

**2. 감질나는 무료 체험**
- 2-3턴으로 가치 체험
- "더 얘기하고 싶다" 감정 유발
- 명확한 등급 차이

**3. 진짜 다른 프리미엄**
- 10-20턴 깊은 대화
- 완전히 다른 수준의 이해
- 128가지 정밀 분석

**4. 즉시 활용**
- 이론만 X
- 복사해서 바로 보낼 메시지
- 실제 문제 해결
